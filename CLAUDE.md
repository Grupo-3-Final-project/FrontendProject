# CLAUDE.md - Proyecto Final Parque de Atracciones

@AGENTS.md

Este proyecto usa `AGENTS.md` como documento principal de normas compartidas para agentes de IA.

Claude Code debe seguir las instrucciones importadas desde `AGENTS.md`.

Antes de modificar cualquier elemento relacionado con la comunicación frontend-backend, Claude Code debe revisar también:

- `docs/API_CONTRACT.md`
- `docs/CONTRACT_TESTING.md`

Esto aplica a:

- Endpoints.
- Métodos HTTP.
- DTOs.
- Requests.
- Responses.
- Nombres de campos JSON.
- Códigos de error.
- Servicios API del frontend.
- Tests de contrato.
- Swagger/OpenAPI.

Si una tarea afecta al contrato API, Claude Code debe indicar qué documentos ha revisado y qué partes del contrato se ven afectadas.
